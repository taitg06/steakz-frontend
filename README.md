# Steakz Restaurant Website

This is a React + TypeScript website for Steakz, a modern steakhouse with an open area layout. The site features:

- **Home:** Welcome message, signature dishes, and open area dining info
- **Menu:** Full menu with steaks, seafood, sides, and desserts
- **About Us:** Restaurant story, philosophy, and location
- **Contact:** Reservation info, hours, and social links

## Getting Started

1. Install dependencies:
   ```
npm install
   ```
2. Start the development server:
   ```
npm start
   ```

## Project Structure
- `src/components/` — Layout and navigation components
- `src/pages/` — Home, Menu, About, and Contact pages

## Customization
Replace placeholder info and images with your own for a personalized experience.
